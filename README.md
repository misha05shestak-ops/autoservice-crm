# autoservice-crm
# CRM Автосервис - Учет и Финансы

Готовая локальная система управления автосервисом (CRM) с графическим интерфейсом. Разработана для автоматизации бизнес-процессов станций технического обслуживания без необходимости платить за облачные подписки.

## Основной функционал
* **Управление заказами:** Полный цикл ведения клиента. Фиксация статусов ("В работе", "Ожидает запчастей", "Выполнен"), расчет итоговой стоимости ремонта и автоматический учет затраченного времени на каждую машину.
* **Складской учет:** Две независимые базы данных:
  * *Запчасти:* Учет расходников с расчетом маржи (цена закупки / цена продажи).
  * *Инструменты:* Учет внутренних активов мастерской.
* **Финансовая аналитика:** Построение графиков выручки по месяцам и круговых диаграмм соотношения доходов (работа/запчасти) в реальном времени.
* **Интеграция и Аналитика:** Двусторонний обмен данными с Microsoft Excel (.xlsx).
* **Печать:** Генерация и прямая печать документов, заказ-нарядов и складских отчетов с финансовыми итогами.

## Технологический стек (под капотом)
* **Язык:** Python 3.13
* **Графический интерфейс:** PyQt6
* **База данных:** SQLite3 (работает локально, полностью автономно)
* **Аналитика и графики:** Pandas, Matplotlib, Openpyxl

## Как пользоваться
Программа портативна и не требует установки.
1. Скачайте файл `crm-installer.exe` из раздела **Releases**.
2. Запустите файл на любом ПК с Windows.
3. Программа автоматически создаст файл базы данных `autoservice.db` в той же папке и будет готова к работе.
